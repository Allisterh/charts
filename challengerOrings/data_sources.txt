The data are records of the number and type of O-ring failures prior to the tragic Challenger mission in January, 1986.

Original data source:
Presidential Commission on the Space Shuttle Challenger Accident, Vol. 1, 1986: 129-131.

The data are stored in the DAAG library, which contains, 'Various data sets used in examples and exercises in the book Maindonald, J.H. and Braun, W.J. (2003, 2007, 2010) "Data Analysis and Graphics Using R".'

Detailed information on the library can be found here:
https://cran.r-project.org/web/packages/DAAG/index.html
